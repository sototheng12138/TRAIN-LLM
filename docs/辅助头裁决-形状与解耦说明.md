# 辅助头裁决：形状与「解耦裁决」说明

## 你期望的完全解耦

- **preds** 形状：`[Batch, Seq_Len, 4]`（4 个品种/通道）
- **aux_probs** 形状：也应为 `[Batch, Seq_Len, 4]`，**点对点、通道对通道**，不跨通道取 max、不跨时间步取 mean/max

## 当前模型辅助头的实际输出

- 在 **TimeLLM** 里，辅助头输入是 **repr_pooled = block.mean(dim=(1, 3))**，即对（通道维 + patch 维）做平均，得到 **每个样本一个向量**。
- 独立通道时，一个 batch 里样本数为 **n_windows * n_channels**（每个窗口拆成 4 条，每条一个通道），所以：
  - **aux_logits** 形状：`(n_windows * n_channels, 2)` → 取 P(有发运) 后为 **`(n_windows, n_channels)`**
- 即：**每个 (窗口, 通道) 一个置信度，没有 Seq_Len 维**。同一窗口、同一通道内的所有时间步共用一个 gate。

## 当前评估逻辑（已按「通道对通道」改）

- **不再**跨通道取 max。
- **gate** 形状：`(n_windows, n_channels)`，与 pred 的通道维逐元素对应。
- **裁决**：`pred = pred * gate[:, None, :]`，即每个 (窗口, 通道) 独立门控；同一 (窗口, 通道) 内 pred_len 步共用同一 gate。
- 满足：**通道对通道**；**时间维**上因模型只给出一窗一通道一个 aux，无法做到「点对点」每时间步独立。

## 若要得到 [Batch, Seq_Len, 4] 的 aux_probs

需要改 **模型**：辅助头不再对序列维做 mean，而是在 **每个时间步**（或每个 patch）上单独做线性层，得到 `(B, Seq_Len, 4)` 的 aux 输出，再在评估里做 `preds = preds * (aux_probs > 0.5)` 的逐点裁决。
