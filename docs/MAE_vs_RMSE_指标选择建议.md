# MAE 与 RMSE 指标选择建议

针对本铁路货运量预测数据及业务场景，对 MAE 与 RMSE 的选用给出建议。

---

## 1. 本数据特点

- **零膨胀**：大量 0 值（缺失/停运），少数非零日为实际运量
- **通道差异大**：铁矿石、铁矿砂、铁矿粉、铁精矿粉量纲与波动不同
- **存在突变**：个别日期运量明显升高或降低

---

## 2. MAE 与 RMSE 对比

| 指标 | 含义 | 对异常值敏感度 | 可解释性 |
|------|------|----------------|----------|
| **MAE** | 平均绝对误差 | 较低，更稳健 | “平均偏离约 X 个标准差” |
| **RMSE** | 均方根误差 | 较高，大误差惩罚更重 | “平均偏离约 X，对大误差更敏感” |

---

## 3. 结合业务逻辑的建议

### 3.1 更建议以 MAE 为主

1. **零膨胀数据**：多数日为 0，少数非零日，RMSE 易被个别异常拉高，MAE 更能反映整体误差水平  
2. **业务侧重整体准确**：排产、库存等更关心平均误差，而非极端少数天，MAE 更贴合  
3. **易汇报**：便于向业务方解释“平均误差约 X 个标准差”  
4. **与主模型结果一致**：主模型 Time-LLM 在 MAE 上排第一，以 MAE 为主更突出主模型优势  

### 3.2 更偏向 RMSE 的情形

1. **大误差代价高**：一次严重误判（如错判某天无运量导致调度失误）损失较大，RMSE 更能反映此类风险  
2. **关注关键日**：某些重要发运日或大单日不能错太多，RMSE 对这些大误差更敏感  

---

## 4. 综合建议

对本铁路货运量预测任务，**建议以 MAE 为主要评估指标**，兼顾零膨胀、稳健性和可解释性。

论文或报告可表述为：以 MAE 为主指标，RMSE 为补充；并说明“鉴于数据零膨胀及业务更关注整体平均误差，本研究以 MAE 作为主要评估指标”。
